Climate visualization using D3.js

Interactive web visualization of global temperature changes using D3.js.
The project presents climate data through charts and maps to highlight long-term global warming trends.

Technologies:
D3.js
JavaScript
HTML/CSS

Visualizations:
Area chart (temperature trends)
Radial and polar charts (temperature anomalies)
Choropleth world map (country-level temperatures)

Data Sources:

HadCRUT4.csv; source: Met Office Hadley Centre & Climatic Research Unit (University of East Anglia); data: global monthly temperature anomalies; period: 1901–2020

temp-1901-2020-all.csv; source: Berkeley Earth; data: monthly average temperatures by country; period: 1901–2020; includes country names
